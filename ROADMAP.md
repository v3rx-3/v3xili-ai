# V3XILI AI Roadmap

This roadmap tracks the public direction of V3XILI AI.

The roadmap is intentionally focused on shipping useful improvements, not promising features before they are ready.

## Current status

V3XILI AI is live at:

https://v3xiliai.online

The current focus is launch feedback, interface polish, stronger AI workflows and preparing the next development direction: kazaAI.

---

## Phase 1 — Public launch and feedback

Status: in progress

Goals:

- Publish V3XILI AI publicly
- Collect feedback from makers, students and developers
- Validate the strongest use cases
- Improve the onboarding and first-use experience
- Keep the website stable on desktop and mobile

Planned work:

- [ ] Collect feedback through GitHub Issues
- [ ] Convert common feedback into small product improvements
- [ ] Improve wording on public pages
- [ ] Add clearer examples for `/humanice` and `/x10think`
- [ ] Improve public documentation

## Phase 5 — kazaAI

Status: early direction

kazaAI is planned as a development agent inside the V3XILI ecosystem.

Main focus:

- long-running coding sessions
- persistent project memory
- CLI-first workflows
- project context continuity
- safe file editing
- debugging and log analysis
- agent-style planning and review

The goal is to help developers continue real software projects without constantly re-explaining the full context.

Planned work:

- [ ] Define public kazaAI scope
- [ ] Prepare CLI concept
- [ ] Prepare memory/checkpoint design
- [ ] Prepare safe file editing flow
- [ ] Collect developer feedback
- [ ] Build first usable prototype
